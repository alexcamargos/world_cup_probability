# World Cup Probability Elo

- Objetivo: estimar a força relativa das seleções em uma escala dinâmica.
- Saída esperada: rating por seleção, data e contexto do jogo.
- Uso no pipeline: feature base para previsão de resultados e calibração de probabilidades.
- Fontes complementares: World Football Elo Ratings e FIFA World Ranking entram
  como features para comparação, inicialização e enriquecimento do treino do
  modelo.
